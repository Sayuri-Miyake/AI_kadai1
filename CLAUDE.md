# CLAUDE.md

## プロジェクト概要

2次元強磁性イジング模型の転移温度をモンテカルロ法で求める計算物理の課題。

## ファイル構成

```
AI_kadai1/
├── CLAUDE.md         # このファイル（Claude Code 向け設定）
├── README.md         # リポジトリ概要
├── overview.md       # 課題の要点（モデル・手法・成果物チェックリスト）
├── background.md     # 背景・歴史的解説（オンサーガー解・MCMC理論・FSS理論）
├── docs/
│   └── worklog.md    # 作業日誌
└── .vscode/
    └── settings.json # VS Code マークダウン数式レンダリング設定
```

## コードスタイル

- インデント：スペース **4つ**
- 1行の最大文字数：**88文字**
- 文字コード：**UTF-8**
- 改行コード：**LF**

## ドキュメント方針

- **overview.md**：課題を解くのに必要な情報のみ。箇条書き中心で簡潔に。
- **background.md**：歴史・理論的背景。読み飛ばし可の補足情報。
- 数式は `$$` をブロック数式・`$` をインライン数式で記述（GitHub / VS Code 1.88+ で表示可）。

## Git ルール

- `master` ブランチへの直接 push は避け、feature ブランチを使う（現在は学習段階のため master 直接も可）
- コミットメッセージは英語で変更内容を明確に記述する
